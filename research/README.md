# Aziky Research

This directory contains implementation research and standalone experiments.
It is evidence and design input, not a statement of supported language or
compiler behavior.

- [`compiler`](compiler): instruction selection, integer peepholes, and x86-64
  microarchitecture studies.
- [`experiments`](experiments): standalone C experiments used to examine code
  generation and performance behavior.
- [Superoptimization](superoptimization.md)
- [SIMD](simd.md) and [AVX SIMD](avx-simd.md)
- [Macro-fusion](macro-fusion.md)
- [GCC peephole optimizations](gcc-peephole-optimizations.md)
