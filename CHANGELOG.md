## 0.1.0-alpha.1 - 2026-07-18

- Prepare the repository for its first open-source publication.
- Rename the compiler package and executable from the development name to
  `aziky`.
- Add conjunctive MIT AND Apache-2.0 licensing.
- Add public contribution, security, governance, support, and CI policies.
- Add the compiler-backed VS Code language extension.
- Make package fixtures and diagnostic paths deterministic on Windows hosts.
- Emit a sectioned Mach-O layout compatible with strict macOS code-signing
  validation.
