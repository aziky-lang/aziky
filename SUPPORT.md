# Support

Aziky is an experimental volunteer-maintained project and does not provide a
commercial support commitment.

- Use repository discussions for usage questions and design conversation.
- Use issues for reproducible bugs and accepted feature proposals.
- Include your Aziky commit, host OS, target triple, command, source reproducer,
  output, and expected behavior.
- Use the private process in [SECURITY.md](SECURITY.md) for vulnerabilities.

Please search existing documentation, issues, and discussions first. General
Rust, operating-system administration, or unrelated toolchain questions may be
redirected to more appropriate communities.
